# Финансовый календарь семьи

Локальное React + TypeScript приложение для контроля семейных доходов, обязательных платежей, просрочек, резервов и свободного остатка.

## Возможности MVP

- экран "Сегодня" с текущим остатком, ближайшим платежом, следующим доходом и проверкой риска;
- финансовый календарь месяца с группировкой платежей между датами доходов;
- CRUD для платежей и доходов;
- статусы оплат и поступлений;
- локальный журнал действий;
- настройки остатков по счетам и резервов;
- экспорт платежей, доходов, календаря и логов в CSV;
- Excel-совместимый экспорт в `.xls`.

Данные хранятся в `localStorage` браузера. Приложение не отправляет данные в облако.

## Запуск

```bash
npm.cmd install
npm.cmd run dev
```

## Проверка

```bash
npm.cmd run build
npm.cmd run lint
```
